# 二叉树层序遍历

给你一棵二叉树的层序序列，其中用 -1 表示空节点，请你输出该二叉树的层序遍历结果，按层分组输出。

在层序序列中，节点按从上到下、从左到右的顺序排列。对于位置 i（从 0 开始编号）的节点，其左子节点在位置 2i+1，右子节点在位置 2i+2。用 -1 表示该位置没有节点。

你需要按照树的层级，将每一层的非空节点值输出在同一行，不同层之间换行分隔。如果某一层没有任何非空节点，则不输出该层。

## 输入格式

第一行输入一个整数 n（1 ≤ n ≤ 10^5），表示层序序列的长度。

第二行输入 n 个整数，表示二叉树的层序序列，其中 -1 表示空节点，其余为正整数表示节点值。

## 输出格式

按层输出二叉树的遍历结果，每一层占一行。同一层的节点值之间用空格分隔。如果某一层没有非空节点，则跳过该层不输出。

## 数据范围

- 1 ≤ n ≤ 10^5
- 节点值为正整数或 -1（表示空节点）
- 层序序列中第一个元素不为 -1（保证树非空）

## 提示

- 可以使用广度优先搜索（BFS）逐层遍历二叉树，也可以直接根据索引计算每个元素所在的层数。
- 对于索引 i 的节点，其所在层数为 floor(log2(i+1))，也可以通过计算每层的起始和结束索引来确定层数。
- 第 k 层（从 0 开始）包含 2^k 个索引位置，起始索引为 2^k - 1，结束索引为 2^(k+1) - 2。
- 注意跳过值为 -1 的节点，只输出非空节点的值。

## 样例 #1

### 输入

```
7
1 2 3 4 5 6 7
```

### 输出

```
1
2 3
4 5 6 7
```

### 解释

这是一棵完全二叉树，所有节点都存在。根节点为 1，第二层为 2 和 3，第三层为 4、5、6、7。每一层都包含全部节点，按层输出即可。

## 样例 #2

### 输入

```
7
1 2 3 -1 -1 4 5
```

### 输出

```
1
2 3
4 5
```

### 解释

根节点为 1，其左子节点为 2，右子节点为 3。节点 2 的左子节点（索引 3）和右子节点（索引 4）均为空（-1）。节点 3 的左子节点为 4（索引 5），右子节点为 5（索引 6）。因此第三层只有 4 和 5 两个节点。
