# 合并两个有序序列

给定两个已经按非递减顺序排列的整数序列，将它们合并为一个新的有序序列。合并后的序列也应保持非递减顺序。

这是一个经典的双指针问题。由于两个输入序列已经各自有序，我们不需要对全部元素重新排序，而是可以利用已有顺序高效地完成合并。具体做法是同时从两个序列的起始位置出发，每次比较当前元素的大小，将较小者放入结果序列，然后移动对应的指针，直到两个序列的所有元素都被处理完毕。

该问题是归并排序中"合并"步骤的核心，也是许多更复杂算法的基础。掌握这一问题对于理解分治策略和双指针技巧至关重要。

## 输入格式

第一行包含两个整数 $n$ 和 $m$，分别表示两个序列的长度。

第二行包含 $n$ 个整数，表示第一个有序序列，按非递减顺序排列。

第三行包含 $m$ 个整数，表示第二个有序序列，按非递减顺序排列。

## 输出格式

输出一行，包含 $n + m$ 个整数，为合并后的有序序列，以空格分隔。

## 数据范围

- $1 \le n, m \le 10^5$
- 序列中的每个元素均为 32 位有符号整数范围内的整数
- 每个输入序列已按非递减顺序排列

## 提示

- 使用双指针方法：分别用指针 $i$ 和 $j$ 指向两个序列的当前位置，比较 $a_i$ 和 $b_j$，将较小者加入结果并移动对应指针
- 当其中一个序列的元素全部处理完后，直接将另一个序列的剩余元素依次加入结果
- 时间复杂度 $O(n + m)$，空间复杂度 $O(n + m)$
- 注意处理一个序列先遍历完毕的边界情况

## 样例

### 样例 1

**输入**

```
3 3
1 4 7
2 5 8
```

**输出**

```
1 2 4 5 7 8
```

**解释**

第一个序列为 [1, 4, 7]，第二个序列为 [2, 5, 8]。合并过程如下：1 < 2，取 1；2 < 4，取 2；4 < 5，取 4；5 < 7，取 5；7 < 8，取 7；取 8。最终结果为 [1, 2, 4, 5, 7, 8]。

### 样例 2

**输入**

```
1 1
5
3
```

**输出**

```
3 5
```

**解释**

第一个序列为 [5]，第二个序列为 [3]。3 < 5，取 3；然后取 5。最终结果为 [3, 5]。
