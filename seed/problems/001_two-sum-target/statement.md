# 两数目标和

给定 $n$ 个整数和一个目标值，在这 $n$ 个整数中找出两个数，使它们的和恰好等于目标值。你需要返回这两个数在数组中的位置（从 1 开始编号）。

数组中同一元素不能使用两次。保证如果存在满足条件的两个数，则它们的下标一定不同。

## 输入格式

第一行包含两个整数 $n$ 和 $target$，分别表示整数的个数和目标值。

第二行包含 $n$ 个整数 $a_1, a_2, \ldots, a_n$，表示给定的整数序列。

## 输出格式

输出一行两个整数 $i$ 和 $j$（$i < j$），表示 $a_i + a_j = target$。如果存在多组满足条件的下标对，输出 $i$ 最小的一组。如果不存在满足条件的下标对，输出 `-1 -1`。

## 数据范围

- $2 \le n \le 10^5$
- $-10^9 \le a_i \le 10^9$
- $-10^9 \le target \le 10^9$

## 提示

可以使用哈希表来优化查找过程。遍历数组时，对于当前元素 $a_i$，检查 $target - a_i$ 是否已经出现在之前的元素中。使用哈希表可以将每次查找的时间复杂度降低到 $O(1)$，从而将整体时间复杂度从暴力的 $O(n^2)$ 优化到 $O(n)$。

## 样例

### 样例 1

**输入**

```
4 9
2 7 11 15
```

**输出**

```
1 2
```

**解释**

$a[1] = 2$，$a[2] = 7$，$2 + 7 = 9$，满足条件。

### 样例 2

**输入**

```
6 10
1 2 3 4 5 6
```

**输出**

```
4 6
```

**解释**

$a[4] = 4$，$a[6] = 6$，$4 + 6 = 10$，满足条件。在数组中，满足 $a_i + a_j = 10$ 且 $i < j$ 的下标对只有 $(4, 6)$。
