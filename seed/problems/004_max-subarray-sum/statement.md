# 最大连续子段和

给定一个包含 n 个整数的数组，请找出该数组中连续子数组的最大和。连续子数组是指数组中一个或多个连续的元素组成的子序列。

例如，对于数组 [-2, 1, -3, 4, -1, 2, 1, -5]，连续子数组 [4, -1, 2, 1] 的和为 6，是该数组所有连续子数组中和最大的。

本题要求你实现一个高效的算法来求解这个问题。直观的暴力方法需要 O(n^2) 的时间复杂度，但对于 n 高达 10^5 的数据规模来说显然无法通过。你需要使用动态规划的思想，将时间复杂度优化到 O(n)。

需要注意的是，数组中的元素可能全部为负数。当所有元素均为负数时，最大连续子段和等于数组中最大的那个元素（即绝对值最小的负数）。

## 输入格式

第一行包含一个整数 n，表示数组的长度。

第二行包含 n 个整数，表示数组中的元素，整数之间用空格分隔。

## 输出格式

输出一个整数，表示最大连续子段和。

## 数据范围

- 1 <= n <= 10^5
- -10^9 <= 数组中的每个元素 <= 10^9

## 提示

- 本题的经典解法是 Kadane 算法，其核心思想是动态规划。
- 定义 dp[i] 表示以第 i 个元素结尾的连续子数组的最大和，则 dp[i] = max(a[i], dp[i-1] + a[i])。
- 最终答案为所有 dp[i] 中的最大值。
- 注意使用 64 位整数（如 C++ 中的 long long）来存储中间结果，避免溢出。

## 样例

### 样例 1

**输入：**

```
8
-2 1 -3 4 -1 2 1 -5
```

**输出：**

```
6
```

**解释：** 连续子数组 [4, -1, 2, 1] 的和最大，为 4 + (-1) + 2 + 1 = 6。

### 样例 2

**输入：**

```
5
1 2 3 4 5
```

**输出：**

```
15
```

**解释：** 数组中所有元素均为正数，因此整个数组就是最大连续子数组，其和为 1 + 2 + 3 + 4 + 5 = 15。
