# 括号序列是否合法

给定一个只包含括号字符 `(`、`)`、`[`、`]`、`{`、`}` 的字符串，判断该括号序列是否合法。

一个合法的括号序列需要满足以下两个条件：

1. 每个左括号必须有一个相同类型的右括号与之匹配；
2. 括号必须以正确的顺序闭合，即后出现的左括号必须先被匹配（后进先出）。

例如，`()`、`[]`、`{}` 都是合法的括号序列；`({[]})` 也是合法的，因为括号正确嵌套且类型匹配。而 `([)]` 虽然左右括号数量相等，但闭合顺序不正确，因此不是合法序列。

本题是栈的典型应用场景。遍历字符串时，遇到左括号就入栈，遇到右括号就检查栈顶是否为对应类型的左括号，若匹配则出栈，否则序列不合法。遍历结束后，若栈为空则说明所有括号都正确匹配。

## 输入格式

输入一行，包含一个只由括号字符 `(`、`)`、``[`、`]`、`{`、`}` 组成的字符串。

## 输出格式

输出一行，如果括号序列合法则输出 `Yes`，否则输出 `No`。

## 数据范围

- $1 \le \text{字符串长度} \le 10^5$
- 字符串仅包含字符 `(`、`)`、`[`、`]`、`{`、`}`

## 提示

- 使用栈来维护尚未匹配的左括号。
- 遇到右括号时，若栈为空或栈顶左括号类型不匹配，则序列不合法。
- 遍历结束后，若栈不为空，说明有未匹配的左括号，序列也不合法。
- 注意时间复杂度应为 $O(n)$，空间复杂度应为 $O(n)$。

## 样例 1

**输入：**

```
()[]{}
```

**输出：**

```
Yes
```

**解释：** 三对括号均正确匹配：`()`、`[]`、`{}`，且闭合顺序正确。

## 样例 2

**输入：**

```
([)]
```

**输出：**

```
No
```

**解释：** 虽然左括号和右括号的数量相等，但 `(` 应与 `)` 匹配、`[` 应与 `]` 匹配，而实际闭合顺序为 `)` 先匹配了 `(`，但中间穿插了 `[` 和 `)`，导致 `(` 先于 `[` 被闭合，违反了后进先出的匹配规则。
